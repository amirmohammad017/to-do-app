# 📝 To-Do App

A simple **To-Do List** built with **HTML, CSS, and JavaScript**.  

---

## ✨ Features
- ➕ Add new tasks  
- ✅ Mark tasks as completed  
- 🗑️ Delete tasks  
- 💾 Save tasks in **Local Storage**  
- 📱 Responsive design
